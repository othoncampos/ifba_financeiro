# Sistema Financeiro IFBA — Contas a Pagar e a Receber

Projeto desenvolvido pelos alunos do **4º ano do Curso Técnico em Informática do IFBA - Campus Paulo Afonso**, sob orientação do prof. Othon Campos.  
O objetivo é criar um **sistema completo de controle financeiro**, com cadastro de pessoas, categorias, contas e parcelas, integrando **frontend Vue 3 + Vuetify**, **backend Java Spring Boot** e **banco de dados PostgreSQL**.

---

## Tecnologias Utilizadas

### Frontend
- [Vue 3](https://vuejs.org/)
- [Vuetify 3](https://vuetifyjs.com/)
- [Pinia](https://pinia.vuejs.org/)
- [Vite](https://vitejs.dev/)

### Backend
- [Spring Boot](https://spring.io/projects/spring-boot)
- Spring Data JPA  
- Bean Validation (Jakarta Validation)  
- [Flyway](https://flywaydb.org/) — controle de migrações  
- PostgreSQL Driver  

### Banco de Dados
- [PostgreSQL 16](https://www.postgresql.org/)
- Docker Compose com `pgAdmin4` para visualização

---

## Estrutura do Projeto
/ (raiz do repositório)
